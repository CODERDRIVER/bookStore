# cron 表达式详解
https://blog.csdn.net/wqh8522/article/details/79224290
一个cron 表达式至少有6个(也可能是7个)由空格分割的时间元素，按顺序依次为：
- 秒
- 分钟
- 小时
- 天
- 月
- 星期（1-7 1=sun）
- 年份
